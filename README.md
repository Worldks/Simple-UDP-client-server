# Owerview  
Программа сервер после запуска "демонизируется", принимает аргументом номер порта на котором будет принимать  
датаграммы. Сервер подсчитывает общий обьем принятых датагармм и их общее количество, возвращая клиенту   
оба счетчика в текстовом виде.  
Программа клиент требует в аргументах ip-адрес и номер порта назначения(сервера), а также размер  
отправляемой датаграммы на сервер. Клиент будет ожидать ответа от сервера 10 секунд(SIGALRM), если ответ не поступит,  
клиент завершит работу и выдаст ошибку.


# Build  
*client*  
>gcc -Wall -g client.c my_string.c -o client  

*server*  
>gcc -Wall -g server.c my_string.c -o server
